A kind of [`input`](input.md) that handles numbers.

## Usage

### Simple

You can create a <i>number</i> field bound to a numerical variable with the following content:

!!! example "Page content"

    === "Markdown"

        ```
        <|{value}|number|>
        ```
  
    === "HTML"

        ```html
        <taipy:number>{value}</taipy:number>
        ```
